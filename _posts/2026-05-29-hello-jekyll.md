这是我的第一篇博客文章！用 Jekyll 搭建博客非常简单：

1. 创建 Markdown 文件
2. 添加 front matter（文件头部的配置）
3. 推送到 GitHub

Jekyll 会自动把 Markdown 转成漂亮的 HTML 页面。

## 为什么选择 Jekyll？

- **简单**：不需要数据库，纯文本文件管理
- **快速**：生成的是静态 HTML，加载速度极快
- **免费**：GitHub Pages 免费托管
- **安全**：没有后台，不用担心被攻击
